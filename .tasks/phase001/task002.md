## 포함 기능

### 기능 1. 로컬 사용자 계정 생성/로그인

- 계정 생성 화면과 로그인 화면 구현
- `user_id`, `display_name`, `device_name`, 비밀번호 입력/검증 흐름 구현
- `Argon2id` 기반 비밀번호 해시 저장

### 기능 2. SQLite 기반 메타데이터 저장소 구성

- `users`, `settings` 테이블 우선 생성
- `drift` 마이그레이션 구조 도입
- 앱 재실행 후 사용자/설정 상태 복원

### 기능 3. 보안 저장소 연동

- `flutter_secure_storage`를 이용해 로컬 마스터 키 또는 민감 설정 저장
- Linux keyring/Secret Service 의존성 고려한 예외 처리
- 민감 정보와 일반 메타데이터 저장 경로 분리
