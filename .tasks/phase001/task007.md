## 포함 기능

### 기능 1. 다중 파일 전송 모델

- 하나의 전송 작업에 여러 파일 포함
- 파일별 메타데이터와 전체 배치 ID 관리
- 동일 배치 내 파일별 성공/실패 추적

### 기능 2. 1:N 전송 오케스트레이션

- 하나의 사용자 작업을 대상별 `TransferSession`으로 분해
- 공통 파일 메타데이터와 해시 재사용
- 특정 대상 실패가 전체 배치를 중단시키지 않도록 설계

### 기능 3. 큐 및 동시 세션 제한

- 동시 세션 수 제한
- 대기열, 실행 중, 완료, 실패 상태 관리
- 취소, 재시도, 우선순위 변경 기초 동작

## 구현 체크리스트

- [ ] 여러 파일을 선택해 하나의 배치로 전송할 수 있다.
- [ ] 여러 대상을 선택해 1:N 전송을 시작할 수 있다.
- [ ] 대상별 상태가 독립적으로 표시된다.
- [ ] 동시 세션 수 제한을 넘는 작업은 대기열로 이동한다.
- [ ] 한 대상 실패가 다른 대상의 완료에 영향을 주지 않는다.
- [ ] 배치 취소와 개별 세션 취소가 구분되어 동작한다.
